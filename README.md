# React courses I have done with Net Feasa
### Frontend Masters: React and TypeScript, v3
#### Steve Kinney

Version three goes beyond v2’s component props, hooks, and context basics with advanced React + TypeScript patterns like discriminated unions, reducer typing, and reusable generics, plus runtime schema validation with Zod and deeper typing for async data, context, and design systems.
##### Objectives
Write higher-quality code with React and TypeScript. Add critical type safety to hooks and component design systems. Learn patterns for complex state management with reducers and discriminated unions. Validate API schemas at runtime with Zod and create a robust type system that catches errors early and gives your team a better developer experience.

[react-typescript-v3](https://github.com/alerebal/courses/raw/main/react-typescript-v3-dark.pdf)

### Frontend Masters: React Performance, v2
#### Steve Kinney

Version two updates the curriculum for React 19 performance strategies (hydration, Suspense, resource loading, and server actions) while expanding the practical optimization toolkit with virtualization, code splitting, and deeper profiling to spot costly re-renders.
##### Objectives
Ship high-performance React applications! Leverage React 19's performance strategies for hydration, suspense, resource loading, and server actions. Reinforce time-tested best practices for memoization, virtualization, and code splitting. Quickly diagnose performance bottlenecks and costly re-renders and build apps that look and feel fast!

[react-performance-v2](https://github.com/alerebal/courses/raw/main/react-performance-v2-dark.pdf)
